Ejercicio 4


¿A qué puerto se reciben normalmente las peticiones del protocolo HTTP? ¿A qué capa del modelo TCP/IP se encuentra el protocolo HTTP? ¿Y los protocolos TCP, UDP, e IP?

Las peticiones del protocolo HTTP normalmente se reciben en el puerto 80 para las conexiones HTTP estándard y en el puerto 443 para las conexiones HTTPS. También se encuentra en la capa 4 o de aplicación del modelo TCP/IP. 

TCP: puerto 80, capa de transporte(la cuarta capa del modelo OSI).

UDP: puerto 161, capa de transporte(la cuarta capa del modelo OSI).

IP: puerto 80, capa de Internet.
